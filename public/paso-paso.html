Crear un dominio local

  1. Abrir el archivo: C:\Windows\System32\drivers\etc\hosts como administrador y en la parte final del archivo escribir.
    127.0.0.1     apparbolgen.test
  2. Guardar y cerrar.
  3. Abri el archivo de texto plano de configuración de Apache C:\xampp\apache\conf\extra\httpd-vhosts.conf.
  4. Ir al final del archivo y anexar lo siguiente:
    • Si nunca has creado un virtual host agregar:
      <VirtualHost *>
    DocumentRoot "C:\xampp\htdocs"
    ServerName localhost
</VirtualHost>
      Nota: Esta estructura se agrega una única vez.
    • Luego agregar:
      <VirtualHost *>
    DocumentRoot "C:\xampp\htdocs\trabajos\apparbolgen\public"
    ServerName apparbolgen.test
    <Directory "C:\xampp\htdocs\trabajos\apparbolgen\public">
        Options All
        AllowOverride All
        Require all granted
    </Directory>
</VirtualHost>
  5. Guardar y cerrar.
  6. Reiniciar el servidor Apache.
    Nota 1: ahora podemos ejecutar nuestro proyecto local en el navegador introduciendo la siguiente dirección: http://apparbolgen.test
    Nota 2: En caso de que no funcione el enlace, cambiar en el archivo C:\xampp\apache\conf\extra\httpd-vhosts.conf todos los segmentos de código <VirtualHost *> por *<VirtualHost :80>.

  1. Diferentes formas para crear modelos:
    • Crear solo el modelo
      • $ php artisan make:model Model
    • Crear el modelo con migración:
      • $ php artisan make:model Model -m
    • Crear el modelo con migración y controlador:
      • $ php artisan make:model Model -mc
    • Crear el modelo con migración, controlador y seeder:
      • $ php artisan make:model Model -mcs
    • Crear el modelo con migración, controlador, seeder y factory:
      • $ php artisan make:model Model -mcsf
    • Crear el modelo con migración con todo:
      • $ php artisan make:model Model -a
